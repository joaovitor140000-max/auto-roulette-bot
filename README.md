# Auto Roulette Signal Bot

Bot profissional de análise estatística e machine learning leve
para Auto Roulette Live (Evolution).

⚠️ Este projeto **NÃO executa apostas**.
Ele apenas fornece **alertas estatísticos**.

- Telegram Bot 24h
- Análise estatística avançada
- Backtest automático
- Machine Learning leve (confirmação)
- Gestão de risco por sessões

- Modo ALERTA / OPERACAO
